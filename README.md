# Calculadora API DevOps

Projeto simples criado para a disciplina de DevOps para a Atividade Somativa. PUC-PR.

## Como executar

npm install  
npm start  

## Endpoints

/calculadora/somar?a=10&b=5  
/calculadora/subtrair?a=10&b=5  
/calculadora/multiplicar?a=10&b=5  
/calculadora/dividir?a=10&b=5
